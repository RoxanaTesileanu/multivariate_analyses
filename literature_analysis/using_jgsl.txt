Using JGSL
Author: Roxana Tesileanu, INCDS BV, roxana.te@web.de

http://jgsl.sourceforge.net/

The JGSL is a collection of Java classes that wrap around the GNU Scientific Library.

Add the jgsl.jar to the lib directory of the Scala project.
Install gsl on Ubuntu with $sudo apt get install gsl-bin libsl-dbg libgsl0dbl-dev libgsl0dbl (https://askubuntu.com/questions/490465/install-gnu-scientific-library-gsl-on-ubuntu-14-04-via-terminal)
